# Folia Onboarding Prototype

Static React/Babel prototype for the Folia onboarding flow.

## Local Preview

From the repository root:

```bash
python3 -m http.server 5173
```

Then open:

```text
http://127.0.0.1:5173/
```

## Deploying

This can be served as a static site on Vercel. The deploy entrypoint is `index.html`, and the image assets live under `uploads/`.
